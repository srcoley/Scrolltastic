Scrolltastic jQuery Plugin

Scrolltastic lets you add scrolling functionality to an element with "overflow: hidden;" applied. With Scrolltastic, you can scroll by setting an "up" and "down" anchor, or by using the mouse wheel on that specific element.

Download

Download Latest

-OR-

You can pull the latest development version using git: git clone git://github.com/srcoley/Scrolltastic.git

Requirements

  • jQuery(duh)
  • jQuery Mousewheel Plugin
  • jQuery Timeout Interval Idle Plugin

Example 1

Bare bones example. Use the mouse wheel to scroll or click & hold the "Up" or "Down" link to scroll, let go when you want to stop scrolling. 

$("#content").scrolltastic({
	up: $("up"),
	down: $("#down")
});

Example 2

This example shows you how versatile Scrolltastic is. Even with images and varying marings, it still works. *It does not play nice with padding though*

/*
 *
 * If images are present in the div
 * you MUST wrap the Scrolltastic
 * code in this $('img').load function
 *
 * You just gotta
 *
 */
$('img').load(function(){
	$("#content2").scrolltastic({
		up: $("#up2"),
		down: $("#down2")
	});
});
